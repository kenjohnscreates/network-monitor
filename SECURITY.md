# Security

## Reporting

Please report sensitive security issues through a private channel to the repository maintainers rather than filing a public GitHub issue when disclosure could aid abuse.

## Sensitive data

Do **not** attach full packet captures, PCAPs, large CSV/JSON exports, screenshots of live traffic logs, or other artifacts that contain **real IP addresses**, hostnames, or credentials to **public** issues. Prefer redacted summaries and generic reproduction steps instead.
